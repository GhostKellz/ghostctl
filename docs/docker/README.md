# Docker Management

GhostCTL provides comprehensive Docker container management.

## Documentation

- [Containers](containers.md) - Container operations and cleanup
- [Compose](compose.md) - Stack management and homelab templates
- [Security](security.md) - Vulnerability scanning and best practices

## Quick Commands

```bash
ghostctl docker menu              # Docker management menu
ghostctl docker install           # Install Docker
ghostctl docker status            # Docker status
ghostctl docker homelab           # Homelab stack templates
```

## Features

- Container lifecycle management
- Image and volume cleanup
- Compose stack management
- Homelab stack templates
- Security scanning with Trivy
- Registry operations
- Resource monitoring
