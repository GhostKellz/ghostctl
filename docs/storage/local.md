# Local Storage Tools

## Overview

GhostCTL provides local storage management tools for disk operations and maintenance.

## Access

```bash
ghostctl storage menu
# Select: Local Storage Tools
```

## Features

- Disk usage analysis
- Filesystem operations
- Storage health monitoring
- Mount management
